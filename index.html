📖 使用说明（点开/收起 · 每个模块都能单独预估）

怎么用：顶部 5 个标签（①头程 ②入库 ③仓储 ④2C出库 ⑤全链路）点一下切换，每个都是独立的小计算器。

  • ① 头程散货：填体积/重量 → 头程运费
  • ② 入库操作：填件数/SKU数/实重 → 入库操作费（可选"我的合同费率"=谷仓实结单价）
  • ③ 仓储费：填体积/库龄/货型 → 仓储费（含免租期/超库龄）
  • ④ 2C出库：填实重/件数/Zone → 出库操作费+尾程运费
  • ⑤ 全链路：汇总上面各项 → 端到端单件总成本

单模块能不能单独预估？ 能。每个标签完全独立，不用把 5 个都填满。只想估出库费就只用 ④，想估入库操作费就只用 ②；只有看"从头程到出库的单件总成本"才用 ⑤ 汇总。

折扣区（页面顶部）：6 个输入框 + 1 个运费滑块，填"实付比例%"。默认是你账号 RVG1039 真实账单反推值，一般不用动；改完立即重算。

两个易错点：① 出库操作费按实重取档，尾程运费按计费重=max(实重,材积重/8000)取档——两套基准别混；② 尾程重量档要向上进位（1.838kg 取 2.27 档，不是 1.81 档）。

头程海运散货（LCL）成本

✅ 本模块可单独预估，无需填写其他标签。计费体积重取大：1 CBM = 500 KG。最小0.1方、每票最低1方。头程报价含清关/拖车/起运杂费；不含税金、保险、查验附加。RMB类费用单列。

入库操作费成本（含 新品SKU / 混装）

✅ 本模块可单独预估，无需填写其他标签。卸货费按体积CBM计；分拣/理货/入库操作费按件或SKU计。混装=来货理货费(箱内SKU≥2)；新品=新品查验费(按货型档)。
⚙ 按重量档细分操作费（可选，更接近实况）
留空则按上方「单件实重」套单一档；填入各重量档件数则按挂牌档位加权（逐项折扣率中的「操作 IB_F 折扣率」仍生效）。合同费率模式忽略此分布。

仓储费（含免租期 / 超库龄）

✅ 本模块可单独预估，无需填写其他标签。公开价：谷仓物流前30天免租，第31天起按日费率；客户物流第1天起算。API真实=实际扣费含超库龄附加费（Feiyue库存实测 USEA≈公开价2.4×）。

2C 出库成本（单品单件 / 单品多件 / 多品多件）

✅ 本模块可单独预估，无需填写其他标签。出库操作费按件计（与SKU数无关）；多件打包费在包裹内件数>1时按包裹总重档收取。多品多件主要额外成本=多件打包费+更长拣货时间（价目表按件计费，无独立多SKU附加）。

端到端单件履约成本模型（头程→入库→仓储→2C出库）

✅ 汇总端到端单件成本（也可单独打开本标签使用）；单件总履约 = 头程分摊 + 入库单件 + 仓储单件/月 + 出库单件(含运费)。仓储按"库龄天数×日费率×体积/件数"分摊到单件。
